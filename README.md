# StandLeo ⚔️

Faceit-платформа для приватных матчей Standoff 2.

## 🚀 Быстрый старт

```bash
cd c:\Users\User\faceitminiapp
node backend\server.js
```

Открой в браузере: **http://localhost:3000**

## 🎮 Как это работает

### 1. Вход
- Введи свой ID из Standoff 2 (только цифры)
- Введи никнейм
- Нажми "Войти"

### 2. Главная
- Профиль (ELO, статистика)
- Кнопка "Играть" для поиска матча
- Лидерборд (топ игроков по ELO)
- Админ-панель (если ты админ)

### 3. Поиск матча
1. Нажми "Играть" → выбери режим 2v2
2. Нажми "Найти лобби" → выбери свободное лобби
3. Жди пока соберутся 4 игрока
4. Подтверди готовность (все 4 должны подтвердить)

### 4. Пик фазa
1. **Капитаны** выбираются случайно (T и CT)
2. **Пик карт** (30 сек): CT → T → CT
   - Карты: Breeze, Dune, Hanami, Prison, Province, Rust, Sandstone
3. **Пик игроков** (30 сек): T → CT (капитаны набирают тиммейтов)

### 5. Матч
- Показывается информация: команды, карты, ID лобби
- Капитан CT создает приватку и дает ID

### 6. Отправка результатов
- Нажми "Отправить результаты"
- Выбери победителя (T или CT)
- Прикрепи скриншот (необязательно)
- Администраторы подтверждают результат

### 7. ELO система
- Начальное ELO: 1000
- K-фактор: 32
- Формула: ELO = K × (1 - 1/(1 + 10^((ELO_соперника - ELO_моего) / 400)))
- Победа над равным: ~+16 ELO
- Победа над сильным: больше ELO
- Поражение слабому: больше потерь

## 🛡️ Админы (ID: 1562788488, 5731264879)

## 🔧 API Endpoints

### Публичные
- `POST /api/auth` - Вход (gameId, nickname)
- `GET /api/profile` - Профиль
- `GET /api/lobbies` - Список лобби
- `POST /api/lobbies/:id/join` - Войти в лобби
- `POST /api/lobbies/:id/leave` - Покинуть лобби
- `POST /api/lobbies/:id/confirm` - Подтвердить матч
- `POST /api/lobbies/:id/not_confirm` - Не подтвердить
- `POST /api/match/:id/pick_map` - Выбрать карту
- `POST /api/match/:id/pick_player` - Выбрать игрока
- `POST /api/match/:id/submit_result` - Отправить результат
- `GET /api/leaderboard` - Лидерборд

### Админские
- `GET /api/admin/pending` - Ожидающие результаты
- `POST /api/admin/confirm/:id` - Подтвердить результат
- `POST /api/admin/make_admin` - Назначить админа

## 📡 WebSocket события

Лобби обновляются в реальном времени через WebSocket.
